Hello GpuApp

GpuApp is a library for eliminating the gotchas and boilerplate from work in the WebGPU.

This is an example space and also expiremental playground to understand the confusing parts about the WebGPU API.

Examples

More details can be found on github. Each section has a directory with a README file, and of course code!

Background attachment

When setting up a pipline, you have the option to run a `clear` operation that will set the background to a color. The color can be transparent, but there are some gotchas.
  • Simple background color. It includes but doesn't use the shader.
  • Background color that changes per frame. It includes but doesn't use the shader.

Rendering similar background with data passed to the GPU

You can work around the transparency issues by rendering in a background in the GPU if you do some unintutive math, premultiplying, in the shader.
